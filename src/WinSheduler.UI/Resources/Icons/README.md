# Icons Directory

Place your PNG icons here (16x16 or 32x32 recommended).

Required files:
- refresh.png
- add.png
- edit.png
- delete.png
- history.png
- folder.png
- save.png
- cancel.png
- ok.png
- trigger.png

After adding icons, set their Build Action to "Resource" in Visual Studio:
1. Right-click each icon file
2. Properties → Build Action → Resource
